# 🫒 Ollive AI Assessment Platform

> A production-quality mini AI inference & evaluation platform comparing open-source vs hosted LLMs.

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://python.org)
[![Streamlit](https://img.shields.io/badge/UI-Streamlit-red.svg)](https://streamlit.io)

Two AI assistants compared:
- 🌿 **OSS**: Qwen2.5-0.5B-Instruct (local, HuggingFace)
- ⚡ **Hosted**: Gemini 2.5 Flash (Google AI API)

## Setup
Add `GEMINI_API_KEY` in Space Settings → Variables and Secrets.
